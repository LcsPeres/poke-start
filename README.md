
# Poke Start

Projeto criado com finalidade de auxiliar os startes da Capgemini.




## Rodando local

Clone o projeto

```bash
  git clone https://github.com/LcsPeres/poke-start
```

Entre na pasta do projeto

```bash
  cd poke-start
```

Instale as dependências

```bash
  npm i --force
```

Inicie a aplicação

```bash
  npm start
```
